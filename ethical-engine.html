Ethics in Motion

Interactive simulation software exploring ethical decision-making in autonomous vehicle scenarios. Developed as undergraduate research and expanded into a distinguished senior thesis project.

Overview

Ethics in Motion is a Python-based simulation platform designed to explore how algorithmic systems make ethical decisions under uncertainty. The system generates realistic dilemma scenarios using configurable templates and demographic probabilities, allowing users to evaluate how decision rules affect outcomes across populations.

The project was designed for classroom use in computer science ethics education at CSB/SJU in CSCI 150 and supports interactive experimentation with decision algorithms.

Python Simulation Data Modeling Ethics

Key Features

  • Template-based scenario generation
  • Demographic probability modeling using census data
  • Custom decision-function testing
  • Detection of “interesting” ethical edge cases
  • Automated statistics and visualization outputs

Technical Stack

  • Backend: Python
  • Frontend: HTML, CSS, JavaScript
  • Data: Census datasets
  • Architecture: modular simulation engine

Research & Academic Impact

This project evolved into a distinguished senior thesis and includes the design of an IRB-approved user study evaluating how interactive simulations influence ethical reasoning in computer science education.

The long-term goal is to better understand how software design choices shape human-centered outcomes in automated decision systems.I hope to publish work
